1. 코드의 재사용성이 높습니다.
  - 새로운 코드를 작성할 때 기존의 코드를 이용하여 쉽게 작성할 수 있습니다.

2. 코드의 관리가 용이합니다.
  - 코드간의 관계를 이용해서 작은 노력으로 쉽게 코드를 변경할 수 있습니다.

3. 신뢰성이 높은 프로그래밍을 가능하게 합니다.
  - 제어자와 메서드를 이용해서 데이터를 보호하고, 올바른 값을 유지하도록 하며, 코드의 중복을 제거하여 코드의 불일치로 인한 오동작을 방지할 수 있습니다.
